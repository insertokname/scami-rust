# *scam***i**-rust

*Software Code Acting as Machine Interface*-**rust** is an interface for the [scamu](https://github.com/insertokname/scamu) emulator built in rust.

see the [S.C.A.M.](https://github.com/insertokname/SCAM) project as a reference. 

Controls:

| nes             | layout 1    | layout 2 |
| --------------- | ----------- | -------- |
| Directional Pad | WASD        | Arrows   |
| A Button        | J           | Z        |
| B Button        | K           | X        |
| Start           | Enter       | C        |
| Select          | Right Shift | V        |
